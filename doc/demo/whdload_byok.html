 Requirements:
 -------------
 This install requires WHDLoad to be in the path.
 It can be found in the WHDLoad package.
 To install from a different drive than DF0: select installation mode Expert.

 The installed intro requires 0.5 MB ChipMem (+0.1 MB OtherMem for preload).
 It can be quit by pressing F10 (key can be changed with Tooltype "QuitKey=").

 The intro requires an installed A500 Kickstart 1.3 image.
 The kickstart image must be located in the directory "Devs:Kickstarts" and
 must be named "kick34005.A500". The image can be extracted from a A500 with
 tools like 'GrabKick' or similar which can be found on aminet.
 If the image is of wrong version or altered in any way WHDLoad will quit with
 an appropriate message.
 Additionally the file "kick34005.A500.RTB" must be installed in the same
 directory as the kickstart image. It can be found in the archive
 "util/boot/skick346.lha" on aminet.


 Features:
 ---------
 - Full load from HD
 - Sprite problems fixed
 - Decruncher (PowerPacker) relocated to fast memory
 - NewIcon (created by me)
 - Source code included as usual
 - Quit option that works on 68000 machines (default key is 'F10')

 And another HQC classic has been patched. Always loved this intro
 and always will, specially the tune (a cover of David Whittaker's
 "Storm" tune) is great.

 The intro uses a few graphics.library routines
 (ScrollRaster, Rectfill etc.) which I didn't feel like recoding hence
 the patch needs a Kickstart 1.3 ROM.

 The intro has also been protected from decrunching just like the
 Silent Service intro. :)
